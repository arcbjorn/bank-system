### Bank (crypto) accounts management system
using Kotlin, Spring Boot, JUnit, Mockk

**Development**

```shell
# open the list of commands
./gradlew tasks

# runs as a Spring Boot app
./gradlew bootRun

# executable jar archive to run with java cli
./gradlew bootJar

# create Docker image
./gradlew bootBuildImage

# run all tests
./gradlew test
```
