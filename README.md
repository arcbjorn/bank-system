### Bank (crypto) accounts management system
using Kotlin, Spring Boot, JUnit, Mockk

**Development**

```shell
# view the list of commands
./gradlew tasks

# run as a Spring Boot app
./gradlew bootRun

# make executable jar archive to run with Java cli
./gradlew bootJar

# create Docker image
./gradlew bootBuildImage

# run all tests
./gradlew test
```
